Chart.js

Chart.js Simple yet flexible JavaScript charting library for the modern web 

                1.Let's break this code down.

                First, we need to have a canvas in our page. 
                It's recommended to give the chart its own 
                container for responsiveness.
                

                    id="myChart"
                


                2.Now that we have a canvas, we can include Chart.js from a CDN.
                

                    src="https://cdn.jsdelivr.net/npm/chart.js"
                


                3.Finally, we can create a chart. 
                We add a script that acquires the myChart 
                canvas element and instantiates new Chart 
                with desired configuration: bar chart type, 
                labels, data points, and options.
                


                    const ctx = document.getElementById('myChart');
                    
                    new Chart(ctx, {
                        type: 'bar',
                        data: {
                        labels: ['Red', 'Blue', 'Yellow', 'Green', 'Purple', 'Orange'],
                        datasets: [{
                            label: '# of Votes',
                            data: [12, 19, 3, 5, 2, 3],
                            borderWidth: 1
                        }]
                        },
                        options: {
                        scales: {
                            y: {
                            beginAtZero: true
                            }
                        }
                        }
                    });
